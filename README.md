# MetaTrace

**MetaTrace** is a comprehensive web-based metadata forensics platform that simplifies the extraction, visualization, and analysis of metadata from various digital files. Designed for both technical and non-technical users, MetaTrace empowers users to verify file authenticity, detect anomalies, and support digital investigations through an intuitive interface.

## 🌐 Live Demo

🔗 [Try MetaTrace Live](https://meta-trace.vercel.app/)

## 🚀 Key Features

- **Multi-Format File Upload**: Supports images, PDFs, and other document formats.
- **Detailed Metadata Extraction**: Integrated with ExifTool for comprehensive metadata analysis.
- **Predictive Insights**: Anomaly detection using machine learning models (Isolation Forest + Groq LLM).
- **Clean and Intuitive UI**: Built for usability by both forensic experts and everyday users.

## 🔧 Tech Stack

- **Frontend**: Next.js, Tailwind CSS
- **Backend**: Python, FastAPI, ExifTool
- **Database**: MongoDB
- **Machine Learning**: Isolation Forest, Groq LLM

## 🎯 Use Cases

- **Journalists**: Validate image authenticity for ethical reporting.
- **Cybersecurity Analysts**: Trace file leaks and detect tampering.
- **Legal Professionals**: Authenticate and verify document origins.
- **Digital Forensics Experts**: Analyze multimedia metadata for investigations.
